What is SharePoint?

Microsoft SharePoint is a web-based collaborative platform that integrates with Microsoft Office, Launched in 2001. SharePoint isn’t a programme it’s a platform and it’s a collection of many very different products and technologies all wrapped up and given a name “SharePoint”.

SharePoint is a server product we don’t install SharePoint on our laptop or desktop because its installed on our back-end system and shared across network. There are some associated programs that can install on our desktop like SharePoint designer and SharePoint workspace.

What are the areas of the SharePoint ?
  1. Sites– SharePoint makes websites, the best part of this we don’t need any special programs and be a Web designer or programmer to make sites.
    Example: We can create websites for our self or for our team, our company.
    We might create websites or we might use SharePoint sites other people have made. We can create and edit pages.

  2. Communities– SharePoint helps working with other people.
    Example: If I and one other person working on the word document same time, SharePoint can let me do this. If I want a Wikipedia editable by hundred people, SharePoint can do it too.
    It can give us shared calendars, shared tasks, shared discussion boards.

  3. Content– SharePoint gives us a place to our content and stuffs instead of saving it in a local folders or shared drive. We can put it all in SharePoint.
    Example: We can create SharePoint List as a database for our PowerApps application.

  4. Search- SharePoint also provide search engine to search our contents and stuffs. It’s a massive search engine that does not allow us to search only our own content but also it will let us search people and this will be done so securely that no can access to anything they shouldn’t.

  5. Insights- SharePoint helps us to bring all the information and data together and not just it brings together but it brings together to understand it better to organise, make sense amounts of contents taking different kind of things like spreadsheets, business, blogs and business Intelligence system and presenting in way like dashboards, scorecards and visual diagrams that automatically updated in the real time with information inside the SharePoint.
    As a SharePoint developer, we are paid to use my brain not to do manual labour and that means I have to take decisions by this insights.

  6. Composites- No platform, no program, no Operating System can know exactly what we need, so SharePoint has capabilities to be extended. The best part of this, we don’t need to be a programmer. Using SharePoint designer and Visio we can build custom workflows and forms without codes and if we do know code, we can do more things.
